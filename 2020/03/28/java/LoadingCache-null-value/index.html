优雅处理guava之LoadingCache NULL值问题

Mar 28 2020

Google的guava是个很好的项目，提供了诸如集合、缓存、并发、String工具类等等，实乃Java开发利器。这里简单说一下LoadingCache使用时，对于不存在的值如何优雅处理。ifeve.com有翻译cache相关的介绍，在这里。
